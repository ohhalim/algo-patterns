# 실전 문제 (Real Problems)

패턴을 배웠다면, 실제 코딩 테스트 기출로 감각을 키우자.

> **학습 흐름**: `algo-patterns/` 에서 원리 파악 → `real-problems/` 에서 실전 문제 풀기

---

## 문제 목록

| 파일 | 패턴 | 수록 문제 (출처) |
|------|------|-----------------|
| `01-sorting.ipynb` | 정렬 | 가장 큰 수, H-Index, 국영수 |
| `02-searching.ipynb` | 이분 탐색 | 입국심사, 숫자 카드 2, 랜선 자르기 |
| `03-two-pointer.ipynb` | 투 포인터 | 두 수의 합, 세 수의 합, 합이 0인 세 쌍 |
| `04-frequency-counting.ipynb` | 빈도수 카운팅 | 완주하지 못한 선수, 위장, 전화번호 목록 |
| `05-sliding-window.ipynb` | 슬라이딩 윈도우 | 최대 연속 부분합, 최소 크기 부분배열, 문자열 내 애너그램 |
| `06-string.ipynb` | 문자열 | 유효한 괄호, 로마자-정수, 가장 긴 팰린드롬 |

## 문제 풀이 방법

1. 해당 패턴 노트북(`algo-patterns/`)을 먼저 훑어본다
2. 이 디렉토리에서 문제를 읽고, **핵심 아이디어 힌트만 보고** 직접 풀어본다
3. 막히면 힌트 → 더 막히면 `# --- 정답 ---` 주석 아래 코드를 확인한다
4. 정답을 봤더라도 **직접 다시 타이핑**하며 이해한다
