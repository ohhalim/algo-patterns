# 최소 커트라인 패턴 트랙

목표: 구현 문제와 기초 알고리즘 문제를 안정적으로 통과하는 최소 패턴을 순서대로 체화한다.

---

## 진행 순서

1. HashMap/Set
2. Two Pointers
3. Sliding Window
4. Stack/Queue
5. Binary Search
6. BFS/DFS

---

## 공통 학습 규칙

1. 하루 1문제만 완료해도 성공.
2. 한 문제 30분 넘기지 않기.
3. 20분 막히면 해설 확인 후 즉시 재구현.
4. 끝나고 3줄 복기:
   - 왜 이 패턴인지
   - 시간복잡도
   - 오늘 실수 1개
5. 복습: D0, D1, D3, D7.

## 완료 기준

1. 문제를 보고 패턴을 10초 안에 말할 수 있다.
2. 템플릿 없이 15분 안에 구현한다.
3. 경계값 3개를 스스로 테스트한다:
   - 빈 입력 또는 최소 입력
   - 답이 없는 경우
   - 중복/극단값 케이스
