# 알고리즘 패턴 연습

Python 기본 문법을 익힌 후, **알고리즘 패턴**을 숙달하기 위한 노트북 모음.

이 10개 노트북을 순서대로 풀면 파이썬으로 기본적인 알고리즘 문제를 풀 수 있게 됩니다.

---

## 학습 순서

| 순서 | 파일 | 패턴 | 문제 수 |
|------|------|------|---------|
| 1 | `01-sorting-basics` | Bubble/Selection/Insertion Sort, sorted + key | 6 |
| 2 | `02-searching-patterns` | Linear/Binary Search, 삽입 위치, 회전 배열 | 6 |
| 3 | `03-two-pointer` | 양 끝 좁히기, slow/fast, 중복 제거 | 6 |
| 4 | `04-frequency-counting` | HashMap, Counter, defaultdict, 빈도수 | 7 |
| 5 | `05-sliding-window` | 고정/가변 윈도우, 부분 배열/문자열 | 5 |
| 6 | `06-string-algorithms` | 문자열 처리, 괄호, 압축, 로마 숫자 | 7 |
| 7 | `07-math-and-bit` | 소수, XOR, 진법 변환, 거듭제곱 | 6 |
| 8 | `08-greedy-basics` | 주식, Kadane, Jump Game, 거스름돈 | 5 |
| 9 | `09-recursion-basics` | 피보나치, 거듭제곱, 계단, 부분집합 | 6 |
| 10 | `10-comprehensive-review` | **패턴 혼합** - 직접 판단해야 함 | 8 |

**총 62문제**

## 문제 풀이 순서

1. 패턴 설명 읽기
2. 예제 코드 실행해보기

4. `pass` 대신 코드 작성
5. 테스트 케이스 통과 확인
